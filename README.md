#### [取得学士学位的条件](http://www.njzikao.com/xuewei/2392.html)  
1. 参加高等教育自学考试且如下五门学位要求课程考试总成绩达到350分以上（含350分）  
计算机及应用专业： 计算机系统结构、操作系统、C++程序设计（面向对象程序设计、面向对象程序设计C++）、数据结构、数据库系统原理（数据库原理）
2. 英语科目成绩达到70分以上(含70分)
3. 毕业论文成绩达到良好以上（含良好）
